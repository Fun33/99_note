每次VPN客戶後，就不能收outlook、輸入CRM的困擾，可透過以下方式處理。

以下為Ray提供的解決方案J

 

注意事項：

1.重開機就要重來

2.先注意自己有線網路還無線網路

3.目的：決定每個東西該走甚麼路

 

step1>開始,執行cmd(要以系統管理員身分執行)

 

Step2>

查出自己現在的網卡號碼(看是在用有線網路還是無線)

route print

 

 

Step3>輸入以下這串，其中紅字對應樓上

>>E-MAIL

route add 192.168.88.0 mask 255.255.255.0 192.168.6.253 metric 1 if 13

>>CRM

route add 192.168.33.0 mask 255.255.255.0 192.168.6.253 metric 1 if 13
